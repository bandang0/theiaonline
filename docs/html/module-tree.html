Module Hierarchy

  • theia: This is theia, a Python package for Gaussian ray tracing in 3D optical setups.
    • theia.helpers: This is the helpers sub-package of theia.
      • theia.helpers.geometry: Geometry module for theia.
      • theia.helpers.interaction: Module to define interaction functions for theia.
      • theia.helpers.settings: Module to initiate all global variables for theia.
      • theia.helpers.tools: Defines some generic functions for theia.
      • theia.helpers.units: Various units for theia.
    • theia.main: Main module of theia, defines the main function.
    • theia.optics: This is the optics sub-package of theia.
      • theia.optics.beam: Defines the GaussianBeam class for theia.
      • theia.optics.beamdump: Defines the BeamDump class for theia.
      • theia.optics.component: Defines the SetupComponent class for theia.
      • theia.optics.ghost: Defines the Ghost class for theia.
      • theia.optics.lens: Defines the Lens class for theia.
      • theia.optics.mirror: Defines the Mirror class for theia.
      • theia.optics.optic: Defines the Optic class for theia.
      • theia.optics.thicklens: Defines the ThickLens class for theia.
      • theia.optics.thinlens: Defines the ThinLens class for theia.
    • theia.rendering: This is the rendering sub-package of theia.
      • theia.rendering.features: Features module or theia, to represent objects as FreeCAD Python features.
      • theia.rendering.shapes: Shapes module for theia, provides shape-calculating for 3D rendering.
      • theia.rendering.writer: Writer module for theia, to write CAD content to files.
    • theia.running: This is the running sub-package of theia.
      • theia.running.parser: Module for the parsing on input data from .tia file.
      • theia.running.simulation: Defines the Simulation class for theia.
    • theia.tree: This is the tree sub-package of theia.
      • theia.tree.beamtree: Defines the BeamTree class for theia.
